# Installation Steps

# source  https://www.youtube.com/watch?v=DF68MNDxVwU
# time 1:13

## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
